# 🛒 Olist Brazilian E-Commerce — EDA & Dashboard

## Overview
End-to-end exploratory data analysis of the 
Brazilian E-Commerce Public Dataset by Olist, 
with an interactive Power BI dashboard.

## Dataset
- Source: [Kaggle — Olist Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- 9 tables, 100K+ orders, 2016–2018

## Key Insights
- 96,096 unique customers across 27 Brazilian states
- Only 3.1% repeat customer rate — low retention
- SP state dominates with 43% of all customers
- Late deliveries score 2.63 avg vs 4.21 for on-time
- health_beauty is the top revenue category at R$1.26M
- 97% of orders successfully delivered

## Tools Used
- Python (pandas, matplotlib)
- Jupyter Notebook
- Power BI Desktop

## Files
- `Olist_Final.ipynb` — full EDA notebook
- `Olist_Dashboard.pbix` — Power BI dashboard

## Dashboard Pages
1. Overview — KPIs, orders over time, status breakdown
2. Customers — geographic distribution, retention rate
3. Products & Revenue — top categories, seller analysis
4. Delivery & Reviews — late deliveries, review scores
